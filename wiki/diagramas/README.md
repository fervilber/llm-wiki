# Diagramas — LLM Wiki

Carpeta de diagramas creados por el agente con Excalidraw.

## Cómo ver estos diagramas

Abre este vault en Obsidian con el plugin **Excalidraw** instalado.
Los archivos `.excalidraw` se renderizan automáticamente como diagramas interactivos.

## Cómo pedir un diagrama al agente

```
«Crea un diagrama de flujo del workflow de procesamiento de fuentes»
«Dibuja la arquitectura del sistema LLM Wiki con todos sus componentes»
«Genera un mapa visual de los conceptos sobre [tema]»
«Convierte este diagrama Mermaid a Excalidraw: [código]»
```

El agente guardará el resultado en esta carpeta.
